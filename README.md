# RCD_E3HDC
HDC platform 

System Requirements:
Vivado 2022.2
Python 3.10
Conda / Miniconda (recommended)

Setup steps:
1. Create a new conda environment: conda create --name e3hdc python=3.10.12
2. conda activate e3hdc
3. Run: ./setup.sh

The required python packages should now be installed for the environment.
